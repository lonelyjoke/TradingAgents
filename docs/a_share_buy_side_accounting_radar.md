# A股买方财报雷达

这份文档解释 TradingAgents 在做 A 股基本面分析时，应该怎样阅读财务报表。

核心原则很简单：财报有滞后性，但它是最可靠的事实底稿。买方研究不能只复述财报，也不能完全脱离财报去讲故事。更好的做法是：用财报确认过去发生了什么，再结合行业景气度、产品价格、订单、产能和估值，推断未来可能发生什么。

## 1. 收入真实吗

收入增长不等于收入质量好。要先判断收入是不是能转化成现金。

重点看：

- 营业收入
- 应收账款
- 应收票据
- 合同资产
- 销售商品、提供劳务收到的现金
- 经营现金流

常见解读：

- 收入增长，同时经营现金流也改善，通常说明收入含金量较高。
- 收入增长，但应收账款和合同资产增长更快，可能说明回款压力上升。
- 项目制、工程类、设备类公司合同资产较高并不一定坏，但需要看后续回款和坏账风险。
- 消费品公司收入增长但收现变差，可能说明渠道压货或信用政策放松。

分析时不要只问“收入增速多少”，还要问：“这笔收入的钱回来了吗？”

## 2. 利润质量硬不硬

净利润是结果，但买方更关心这个利润能不能持续。

重点看：

- 毛利率
- 经营利润率
- 净利率
- 销售费用率
- 管理费用率
- 研发费用率
- 财务费用率
- 投资收益
- 公允价值变动收益
- 营业外收入
- 资产减值损失
- 信用减值损失
- 扣非净利润

常见解读：

- 毛利率上升，要区分是产品涨价、成本下降、产品结构改善，还是会计口径影响。
- 净利润增长，但主要来自投资收益、公允价值变动或营业外收入，主业质量要打折。
- 扣非净利润弱于净利润，说明利润中可能有一次性因素。
- 研发费用提升不一定坏，但如果大量研发资本化，要警惕利润被短期美化。
- 财务费用上升，可能说明债务成本、汇兑或资金压力正在影响利润。

分析时不要只问“赚了多少钱”，还要问：“这些利润是不是主业赚来的？”

## 3. 未来业绩有没有会计线索

财报虽然是过去式，但资产负债表里有很多未来线索。

重点看：

- 合同负债
- 预收款项
- 合同资产
- 存货
- 预付款项
- 应付款项
- 在建工程
- 固定资产
- 资本开支

常见解读：

- 合同负债和预收款增加，可能说明未来收入可见度增强，尤其适用于软件、工程、设备、白酒、教育服务等商业模式。
- 对现货型周期品公司，合同负债的指示意义可能较弱，要更多看产品价格、库存和供需。
- 存货增加可能是主动备货，也可能是需求变差，需要结合行业价格、订单和库存周期判断。
- 预付款增加可能是锁定原材料，也可能说明上游议价能力强、公司现金被占用。
- 在建工程和资本开支增加，可能代表未来产能释放，也可能意味着行业即将供给过剩。

分析时不要只问“现在利润如何”，还要问：“财报里有没有提前埋下未来收入或成本变化的线索？”

## 4. 资产负债表能不能扛周期

好公司也可能被周期拖垮。尤其是周期、制造、地产链、航运、资源和新能源公司，资产负债表非常重要。

重点看：

- 货币资金
- 短期借款
- 一年内到期的非流动负债
- 长期借款
- 应付债券
- 有息负债
- 净现金或净负债
- 流动资产
- 流动负债
- 营运资本
- 商誉
- 资产减值准备

常见解读：

- 低估值公司如果债务压力很大，可能是价值陷阱。
- 净现金公司在行业低谷时更有安全边际，也更容易熬到周期反转。
- 商誉占比高的公司，要特别注意未来减值风险。
- 短债压力高、现金不足、经营现金流弱，是危险组合。

分析时不要只问“估值便不便宜”，还要问：“它能不能活到景气反转？”

## 5. 股东回报是否匹配

价值投资最终要落到股东回报。

重点看：

- 分红率
- 股息率
- 回购
- 增发和摊薄
- 少数股东损益
- 归母净利润
- 自由现金流
- 资本开支需求

常见解读：

- 高股息要看自由现金流能不能支撑，不能只看静态股息率。
- 回购要看价格是否合理，以及是否只是对冲股权激励摊薄。
- 净利润增长但归母净利润增长弱，说明上市公司股东拿到的利润可能没那么多。
- 高资本开支公司短期分红能力弱，但如果扩产能带来高回报，可能仍然值得。

分析时不要只问“公司赚不赚钱”，还要问：“这份利润最后能不能变成股东回报？”

## 6. 如何和行业景气结合

财报回答“已经发生什么”，行业景气回答“未来可能发生什么”。买方分析要把两者拼在一起。

可以这样看：

- 财报改善，行业景气也向上：趋势可能较强，但要看估值是否已经透支。
- 财报改善，行业景气转弱：警惕高点利润，可能是周期尾声。
- 财报很差，行业景气开始修复：可能存在预期差机会。
- 财报很差，行业景气继续下行：低估值也可能是价值陷阱。
- 财报一般，但合同负债、订单、产品价格、现金流出现边际改善：值得跟踪。

系统应避免两种极端：

- 完全遵从财报：容易变成静态分析，市场可能已经交易掉了。
- 完全脱离财报：容易变成讲故事和投机。

更好的做法是：以财报为事实锚，以行业景气为推演方向，以估值和赔率决定仓位。

## 7. Agent 输出时应该怎么用

基本面 Agent 不需要把每个会计科目都列出来。它应该只回答最影响投资判断的问题：

- 哪些财报事实支持核心看多或看空逻辑？
- 哪些会计项暗示未来业绩可能变化？
- 哪些指标说明利润质量不够硬？
- 哪些指标说明资产负债表存在压力？
- 哪些信息只是线索，还不能当成确定事实？
- 哪些数据需要在下一期财报、公告或行业数据中验证？

一句话总结：

> 财报不是结论，而是投研推理的地基。优秀买方分析师要从会计项里读出经营现实，再用行业景气和估值赔率判断是否值得下注。
