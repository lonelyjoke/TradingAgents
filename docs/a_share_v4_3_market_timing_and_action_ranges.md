# TradingAgents v4.3 大盘情绪校准与操作区间说明

v4.3 新增两个能力：

1. 根据大盘和板块状态校准单一标的评级。
2. 根据偏多或偏空观点，给出更实用的止盈/减仓区间或进场观察区间。

这版的目标不是让系统“预测最低点和最高点”，而是让报告更接近真实投资决策：同样一家公司，在牛市、熊市、极度悲观、极度乐观环境下，应该有不同的仓位和价格纪律。

## 1. 新增工具

新增工具：

```text
get_market_timing_context(ticker, curr_date, look_back_days=120, years=5)
```

它会读取主要指数的：

- 近一段时间涨跌幅。
- 距离阶段高点的回撤。
- 当前点位在观察窗口里的位置。
- 指数 PE TTM 历史分位。
- 指数 PB 历史分位。
- 换手率。

主要观察指数：

- 上证指数。
- 沪深300。
- 中证500。
- 创业板指。

## 2. 市场情绪分层

v4.3 会把市场环境粗略分为五类。

| 市场状态 | 特征 | 评级影响 |
| --- | --- | --- |
| 极度悲观 / 低估市场 | 指数位置和估值分位都很低 | 不应机械悲观；对优质低估标的可适度积极 |
| 偏熊 / 风险偏低迷 | 指数位置低或回撤明显 | 评级偏保守，除非公司质量和估值足够强 |
| 均衡市场 | 没有明显极端情绪 | 公司和板块证据是主要评级依据 |
| 偏牛但风险升高 | 指数强或估值偏高 | 可顺势，但要提高止盈和风控纪律 |
| 极度乐观 / 拥挤市场 | 指数位置和估值分位都很高 | 不应机械乐观；高估值高 beta 标的要更保守 |

## 3. 为什么不能机械处理牛熊

市场情绪不能简单理解成：

```text
牛市 = 全部上调评级
熊市 = 全部下调评级
```

更合理的做法是结合标的特性：

| 标的类型 | 极度悲观时 | 极度乐观时 |
| --- | --- | --- |
| 高质量低估值公司 | 可以更积极寻找分批进场机会 | 如果估值修复充分，考虑逐步止盈 |
| 高 beta 成长股 | 仍需警惕流动性和估值杀伤 | 容易过度拥挤，应提高减仓纪律 |
| 周期股/商品股 | 看产品价格和库存是否见底 | 看商品价格是否高位滞涨 |
| 航运股 | 看运价指数和航线景气是否见底 | 看 TCE/指数是否已处高位回落 |
| 高负债公司 | 低价不一定是机会 | 乐观市场也不能忽略偿债风险 |
| 稳定分红公司 | 极端悲观时可能有防守价值 | 极端乐观时弹性可能落后 |

因此，v4.3 的原则是：

```text
先看公司，再看行业，再看市场；
市场情绪只做校准，不替代公司研究。
```

## 4. 偏多时的止盈/减仓区间

如果最终观点偏多，例如 Buy 或 Overweight，系统应给出：

```text
Profit-Taking / Trimming Range
```

这个区间可以来自：

- 历史估值分位的高位区域。
- 技术压力位。
- 同行估值对比。
- 市场情绪是否过热。
- 商品价格或航运景气是否高位。

例子：

```text
如果股价进入历史估值 70%-80% 分位，同时大盘处于极度乐观区间，可考虑分批止盈。
```

## 5. 偏空时的进场观察区间

如果最终观点偏空，例如 Underweight 或 Sell，系统应给出：

```text
Entry / Re-entry Watch Range
```

这个区间可以来自：

- 历史估值分位回落到低位。
- 技术支撑区。
- 股价回撤后风险收益改善。
- 大盘从极度乐观转向均衡或悲观。
- 周期变量出现企稳迹象。

例子：

```text
如果估值回到历史 20% 分位以下，且商品价格或行业景气不再恶化，可重新纳入观察。
```

## 6. 最终报告新增内容

v4.3 会让最终组合经理报告新增或强化以下内容：

- `Market Regime Adjustment`：市场环境如何影响评级。
- `Profit-Taking / Trimming Range`：偏多时的止盈或减仓区间。
- `Entry / Re-entry Watch Range`：偏空时的进场观察区间。
- `Unverified Key Assumptions`：缺少证据但对结论重要的假设。

## 7. 使用限制

v4.3 给出的区间不是精确预测。

它更像投资纪律：

- 涨到哪里要开始降低贪婪。
- 跌到哪里值得重新研究。
- 什么市场环境下应该更激进。
- 什么市场环境下应该更保守。

如果缺少技术位、估值位或市场证据，系统应该写“情景区间”或“待核验”，不能强行给精确价格。
