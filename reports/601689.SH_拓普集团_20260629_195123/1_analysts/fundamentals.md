# 投资备忘录：拓普集团（601689.SH）

**日期：2026年6月29日**
**分析师：买方基本面研究部**

## 一、公司商业模式与盈利机制

### 1.1 业务架构

拓普集团是中国最大的平台化汽车零部件供应商之一，主营八大产品板块：NVH减震系统、内外饰系统、轻量化车身、智能座舱部件、热管理系统、底盘系统、空气悬架系统、智能驾驶系统。此外，公司已成立机器人执行器事业部，并披露液冷服务器首批订单15亿元，以及车规级制氧新业务。

**收入公式**：营收 = Σ（配套车型产量 × 单车配套价值） + 新业务零星收入。单车配套价值约3万元，且仍在扩展。

**利润公式**：毛利 = 营收 - 直接材料（约60-70%）- 人工- 制造费用；研发费用约占收入5%刚性支出；营业利润再减去财务费用（近年明显攀升）。

### 1.2 行业定位

公司处于汽车产业链Tier 1位置，客户覆盖国际国内智能电动车企和传统OEM。核心竞争壁垒在于：①产品平台化（8大系统覆盖，增强客户粘性和切换成本）；②全栈自研与垂直整合（机械、电控、电机）；③全球化产能布局（波兰、泰国）；④深度绑定头部新能源客户。

### 1.3 行业KPI核查清单

| KPI层 | 验证状态 | 核心发现 |
|-------|---------|---------|
| 客户/车型敞口 | 部分验证 | 前五大客户未披露具体收入占比，但年报提及"与全球创新车企合作不断扩大" |
| 产品利润池 | 部分验证 | 分产品收入/毛利率未按八大板块拆分披露，仅合并主营业务总额 |
| 定价/年降 | 缺失 | 年降条款、原材料传导机制未量化披露 |
| 产能/利用率 | 部分验证 | 泰国工厂2026年上半年投产，波兰二期筹划扩建，无具体利用率数据 |
| 成本/毛利归因 | 部分验证 | 毛利率从FY2024的20.8%降至FY2025的19.4%，Q1 2026为19.3%，但未拆分量价成本因素 |
| 营运资本/现金 | 已验证 | OCF/NI=0.99（Q1 2026），现金转化尚可；应收账款周转稳定 |
| 再投资/ROIC | 部分验证 | 2025年capex约35亿，ROIC=1.95%（Q1 2026年化），投入产出待验证 |
| 第二曲线 | 缺失 | 机器人/液冷/制氧无分业务收入、毛利率、订单细节 |

**结论**：核心汽车零部件业务的财务可见度足够，但新兴业务的量化数据严重不足，是下修信心的重要因素。

## 二、核心业务引擎分析：汽车零部件

### 2.1 需求端 → 供给/产能 → 量价/利用率

**需求侧**：全球乘用车销量温和增长。2025年上半年全球乘用车销售约3008.7万台（同比+4.9%），中国新能源乘用车约645.4万台（同比+37.5%，渗透率47%）。拓普的核心客户是头部新能源车企，受益于新能源结构性增长，但若全球汽车销量放缓（尤其欧美），公司增速将承压。

**供给侧**：公司仍在扩张产能——泰国一期2026年投产，波兰二期筹划扩建。2025年固定资产150.5亿元（同比+10%），在建工程持续。量产新工厂存在爬坡成本，短期内可能压制毛利率。

**量价关系**：公司通过产品线扩展（从单一NVH到8大系统）提升单车配套金额（目前约3万元），部分对冲OEM年降压力。但FY2025毛利率同比下降1.37pp（从20.8%→19.4%），Q1 2026毛利率继续同比微降0.64pp，说明年降与成本压力仍未完全消化。

### 2.2 财务传导链

**收入**：FY2025营收295.8亿元（同比+11.2%），FY2024为266.0亿元（同比+35%），增速明显放缓。Q1 2026单季度营收66.3亿元（同比+14.9%），环比改善但基数效应要考虑。

**毛利率**：FY2024 20.8% → FY2025 19.4% → Q1 2026 19.3%，持续压缩。原因包括：①OEM年降；②原材料（铝、钢、塑料）价格波动；③海外工厂初期爬坡的低效成本；④产品结构变化（低毛利品类增速快于高毛利品类）。

**费用端**：研发费用率维持在5%左右（刚性）；管理费用率2.6%（稳定）；销售费用率0.9%（稳定）。但**财务费用率大幅恶化**——Q1 2026财务费用9644万元，占收入比重1.455%，而Q1 2025仅为0.099%（同比+1.36pp）。这是净利率从9.8%降至8.3%的核心拖累因素。

**净利润**：FY2025归母净利润27.79亿元（同比-7.4%），Q1 2026归母净利润5.52亿元（同比-2.4%）。连续四个季度利润同比负增长。

### 2.3 资产负债表前瞻信号

**合同负债**：Q1 2026末2675万元，环比+27%（FY2025末2106万元），同比-16%（Q1 2025末3201万元）。绝对值很小，不构成强需求信号。

**应收账款**：Q1 2026末56.85亿元，占年化收入比21.5%，环比改善（FY2025末25.0%），说明回款有所加快。

**存货**：43.43亿元，占年化收入比16.4%，环比略升但同比稳定，未见异常积压。

**净现金**：9.56亿元（货币资金40.2亿 - 短期借款28.6亿 - 长期借款2.0亿），虽有净现金但覆盖率不高。总负债率40.8%，可控但短期借款从9.3亿增至28.6亿（YoY），是财务费用飙升的主因。

**商誉**：3.4亿元（占总资产0.82%），来自收购芜湖长鹏等，规模可控。

### 2.4 现金流质量

Q1 2026经营现金流5.47亿元，OCF/净利润=0.99，现金转化率良好。FY2025全年经营现金流44.8亿元（OCF/NI=1.61），因营运资本释放。但资本开支持续高企（FY2025约35亿），自由现金流压力存在。2025年FCF约为9.85亿元。

## 三、新兴业务：第二曲线的期权价值

### 3.1 机器人执行器

公司已成立机器人执行器事业部，处于产能筹备和研发阶段。按照Knowledge Planet行业数据（KPE02-KPE04），特斯拉Optimus和国内人形机器人厂商的Tier 1进展是核心催化剂。目前公司未披露任何定点合同或订单金额，无法计入核心估值。

**当前处理**：零收入归入期权价值，仅支持场景上行情景。

### 3.2 液冷服务器

2025年报披露获得首批订单15亿元，但交付时间、毛利率、客户构成均未披露。液冷赛道竞争激烈，公司缺乏数据中心客户积淀。

**当前处理**：scenario upside计入bull情景，base情景假设2027年起逐步贡献少量收入。

### 3.3 车规级制氧

年报提及"有序落地推进"，无任何收入或订单数据，概念阶段。

## 五、估值与预期差分析

### 5.1 当前市场定价内涵

- 市值：909.8亿元
- PE TTM：32.9x（5年27%分位，相对低位）
- PB：3.7x（5年8.3%分位，历史低位）
- PS TTM：3.0x（5年6%分位，历史低位）

**估值解构**：过去6个月股价下跌27.2%，EPS proxy仅+1.2%，PE压缩-28.1%。这是一轮**估值收缩引发的下跌**，而非盈利恶化。

### 5.2 隐含盈利预期

当前PE TTM 32.9x，如果对标合理PE 25x（比行业中位数31.2x略低，考虑盈利下滑风险），隐含盈利约为36.4亿元——远高于当前28亿的基准盈利。市场似乎仍在**定价过去的高增长预期，而非当下盈利现状**。

如果恢复至行业中位数PE 31x，隐含盈利约29.4亿元，接近base case 2026E的28.1亿元。

### 5.3 预期差

**积极预期差**：市场可能过度定价了盈利进一步下滑的风险（PB处于5年8%分位），如果毛利率在2026H2企稳+财务费用不再恶化，当前估值有修复空间。

**消极预期差**：当前32.9x PE对应的是历史上高增长（20%+ROE）时期估值，而当前ROE已从18%降至12.7%（FY2025），盈利负增长。如果市场认识到增速中枢下移，PE可能继续收缩至25x以下。

## 六、核心投资假设与证伪条件

### 6.1 核心判断

**当前阶段**：高估值/低景气度（PE处于历史低位但绝对水平仍高，盈利负增长）。属于"左侧观察而非右侧确认"阶段。

**核心押注**：公司能否在2026-2028年通过①海外产能释放（泰国/波兰）、②产品结构升级（底盘/热管理/空悬等高附加值品类占比提升）、③新业务（液冷/机器人）商业化，实现毛利率从19.3%回升至20.5%+，从而在当前低PB水平上驱动估值修复。

### 6.2 关键支撑证据

1. **收入增速保持正增长**：Q1 2026营收同比+14.9%，在全球车市温和背景下仍好于行业平均，说明客户份额在提升。
2. **行业结构性增长**：中国新能源渗透率向50%靠近，公司配套车型结构受益。
3. **PB处于历史极低分位**（8.3%），反映极端悲观预期。
4. **经营现金流健康**：OCF/NI≈1.0，利润向现金转化无重大问题。
5. **合同负债环比增长27%**：虽绝对量小，但方向积极。

### 6.3 关键负面证据

1. **财务费用率同比+1.36pp**：短期借款从9.3亿暴增至28.6亿，融资租赁售后回租频繁，隐含现金流压力。
2. **毛利率连续四个季度同比下降**：从FY2024的20.8%降至Q1 2026的19.3%。
3. **归母净利润连续四个季度同比负增长**：FY2025 -7.4%, Q1 2026 -2.4%。
4. **新业务无量化进展**：机器人、液冷、制氧均无收入或订单细节。
5. **相对走势极弱**：过去120天跑输行业等权指数22.5pct，跑输中证500指数44.3pct。
6. **股价呈下行趋势**：20日跌幅16%，日均波动率3.35%，成交量比值0.92说明无抄底资金涌入。

### 6.4 证伪条件

1. **毛利率在2026H1继续同比下滑**：如果2026半年报毛利率低于19%（当前19.3%），说明成本传导和客户年降压力超出预期。
2. **财务费用率继续大幅上升**：Q2 2026财务费用若超过1.5%收入比，将严重侵蚀利润。
3. **营收增速降至个位数**：意味着核心客户订单转弱或市场份额丢失。
4. **经营现金流转负**：OCF/NI连续季度低于0.7。
5. **泰国工厂投产严重延迟或亏损超预期**。
6. **新业务无任何里程碑落地**（2026全年无机器人定点、液冷订单无收入确认）。

## 八、结论

**公司质量**：良好但面临挑战。拓普是国内领先的平台化零部件供应商，产品覆盖广、客户粘性强、研发投入持续。但毛利率持续压缩、财务费用飙升、资本开支高企，使盈利质量存疑。

**当前赔率**：赔率中性偏正。PB处于5年8.3%分位，反映极端悲观定价。但如果毛利率持续下降或财务费用继续恶化，估值仍有下行空间至PB 3.0x（接近5年最低2.99x）或PE 25x。

**配置建议**：**中性观望（Positive Watch）**。当前不宜增配或减配，核心原因是毛利率趋势和财务费用走势尚未出现拐点信号。建议将2026年半年报（预计8月底）作为关键验证节点——若毛利率企稳、财务费用环比改善、海外工厂收入贡献开始体现，可转向积极。

**催化剂路径**：①2026半年报毛利率环比改善→看多信号；②大客户新车型定点公告→提振营收可见性；③机器人执行器获得实质定点→期权价值兑现；④泰国工厂投产效率超预期→毛利率回升。
