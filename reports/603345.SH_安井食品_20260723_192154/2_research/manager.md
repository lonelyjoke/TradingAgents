# Research Manager Accepted Underwriting Model

**Rating**: Overweight

**Research Readiness**: partial

**Core Bet**: 市场对安井食品2026年盈利弹性系统性低估。核心赌注是：在消费弱复苏背景下，公司凭借品类升级（烤肠、虾滑、清真）、成本小幅回落及规模效应，将实现23.6%的净利润增长（16.8亿元），远超市场隐含的约12.5%增长，当前19x TTM PE及17.3x 2026E PE未能充分反映这一改善。半年报（8月底）是关键的催化剂，验证毛利率>23%及收入增速>12%即可启动重估。

### 本报告要回答的关键问题

1. Q1: 2026年收入增长驱动力是否可持续？终端需求、渠道补库与新品放量的贡献几何？
2. Q2: 毛利率修复能否持续？成本顺风、提价和产品结构升级各自贡献多大，面临何种风险？
3. Q3: 子公司（新宏业/新柳伍/鼎味泰）是利润拖累还是增量，对归母净利润的实际影响？
4. Q4: 当前19x PE定价是否充分反映了盈利改善，预期差在何处？
5. Q5: 与其他食品龙头（如天味食品）相比，安井的相对配置价值如何？

### 核心问题裁决

**Q1：Q1: 2026年收入增长驱动力是否可持续？**
- **为何关键：** 收入增速决定利润弹性的基数，也是估值能否从防御性向增长性切换的关键。
- **当前结论：** 部分证实，增长具有持续性，但下半年基数走高可能使增速回落至12%左右。Q1合同负债同比+46.5%（EV018）、Q2淡季主业出货同比10-15%（KPE01/KPE03），以及虾滑+37%等新品放量（KPE09），均指向真实终端动销而非渠道压货。但社零餐饮仅+1.2%、食品CPI-1.6%限制了总量空间，下半年内生增速或放缓至8-10%。
- **采用证据：** EV018: 2026Q1合同负债同比+46.5%；KPE01: 渠道反馈6月主业增速10-11%，Q2淡季10-15%；KPE03: 6月收入双位数增长，淡季好于同期；KPE09: 虾滑品类增速+37%；EV011: 2025年营收161.93亿，2026E 182.4亿（+12.6%）基于渠道和行业趋势
- **最强反证：** Q1合同负债环比下降29%为季节性回款（熊方观点），且Q2环比增速较Q1的30%+已显著放缓，叠加社零疲弱，全年能否维持12.6%存疑。鼎味泰并表贡献约5-8pp，剔除后有机增速可能仅22-25%。
- **模型/估值影响：** 维持2026E营收增速12.6%（182.4亿），未采纳牛方14%或熊方11%的极端假设。收入敏感性：每下降1%增速影响净利润约1.2%，但对估值影响有限，因利润弹性主要来自毛利率。
- **置信度与下次验证：** medium；2026年半年报（2026-08月底）披露Q2营收及同比增速，以及分产品收入构成；若营收增速>15%或新品占比超预期，可上调增速；若<10%，则下调至11-12%。

**Q2：Q2: 毛利率修复能否持续？**
- **为何关键：** 毛利率是利润弹性的核心来源，也是多空双方分歧最大的变量，影响EPS和估值中枢。
- **当前结论：** 毛利率改善趋势确立，但幅度及持续性取决于成本走势和产品结构。2026Q1毛利率24.99%（+1.66pp YoY，EV013），公司Q1、Q2均有小幅提价（投资者互动），且费用率持续收缩。但原材料期货价格分化（生猪/玉米/棕榈油下跌，豆粕上涨），成本端顺风贡献了大部分改善，新品（虾滑、烤肠）毛利率可能低于传统火锅料，结构升级的正面影响被部分抵消。维持全年23.2%的基线，较2025年21.6%明显改善，但低于Q1峰值的25%。
- **采用证据：** EV013: 2026Q1毛利率24.99%；投资者互动（2026-05-28）: Q1部分产品提价，Q2维持政策；EV038-EV041: 生猪/玉米/棕榈油期货同比下跌，豆粕上涨；KPE01: 利润增速快于收入，渠道反馈印证利润弹性
- **最强反证：** 若下半年生猪/豆粕/鱼糜价格反弹，成本顺风逆转，毛利率或降至22%附近；新品毛利率较低可能拖累综合毛利率（熊方观点）。卖方报告（KPE04）提及成本压力，但并未量化。
- **模型/估值影响：** 维持2026E毛利率23.2%（牛方24%未采纳，熊方22.5%未采纳）。毛利率每变动0.5pp约影响净利润4-5%，对每股价值影响约3-4元。
- **置信度与下次验证：** medium；半年报毛利率及分产品毛利率，若H1毛利率≥24%则全年有望达24%以上，若<23%则风险加大；需同步跟踪生猪/豆粕期货价格。

**Q3：Q3: 子公司利润贡献如何？**
- **为何关键：** 子公司（特别是新柳伍的亏损风险）是多头论点的潜在风险点，也影响归母净利润和商誉压力。
- **当前结论：** 新宏业表现超预期（收入利润双位数增长，KPE01），新柳伍收入正增长但利润承压，鼎味泰并表贡献增量但或有整合成本。整体来看，子公司合计有望实现小幅正贡献（约0.3-0.8亿），不会构成重大拖累。商誉7.55亿规模有限，减值风险可控。
- **采用证据：** KPE01: 新宏业Q2收入利润双位数以上，新柳伍收入正增长净利润承压；EV011: 2025年归母净利润13.59亿，包含了子公司部分；资产负债表中商誉7.55亿（占比低）
- **最强反证：** 若新柳伍下半年因小龙虾价格波动或需求不振导致亏损扩大，或鼎味泰整合成本高于预期，可能拖累利润0.5-1亿元；此外，少数股东损益如果增速超过收入，归母利润将被稀释。
- **模型/估值影响：** 在基线16.8亿归母净利润中已包含子公司小幅正贡献（约0.5亿），未单独上调。若半年报证实新宏业继续增长且新柳伍改善，可小幅上调子公司利润预期。
- **置信度与下次验证：** medium；半年报少数股东损益及子公司经营数据披露，若少数股东损益同比增速低于30%，则拖累有限。

**Q4：Q4: 当前19x PE是否充分定价？**
- **为何关键：** 判断是否存在预期差，决定了投资是否具备安全边际和收益空间。
- **当前结论：** 市场定价了谨慎预期，存在可观的预期差。当前市值290亿对应的TTM利润约15.3亿（隐含+12.5%增长），而我们的基线预测2026E净利润16.8亿（+23.6%），隐含的2026E PE仅17.3x，低于历史中枢和历史同水平价格时的PE（17.55x）。若毛利率和收入兑现，估值有修复至18-20x PE的动力，叠加分红（约3.3-5.0%的区间），一年预期回报约15-20%。
- **采用证据：** Market-expectation context: PE TTM 19.0x, 隐含TTM盈利15.3亿；Historical PE-EPS decomposition: 当前PE处于39.2%分位，EPS代理处于59.1%分位，价格受估值压缩拖累；Dividend context: 股息率3.3-5.0%，分红稳定；EV011: 2025年归母净利润13.59亿，2026E 16.8亿
- **最强反证：** 若实际利润仅15-16亿（熊方情景），当前PE将升至18-19x，无安全边际；且市场情绪恶化可能将估值压缩至15-17x，对应股价下跌20-30%。宏观疲弱和成本反弹可能使该情景概率上升。
- **模型/估值影响：** 公允价值采用19x 2026E核心利润+期权价值的SOTP，得出约96-102元/股，较现价有10-17%上行空间。若情景转向乐观，可达144元。基线预期回报吸引。
- **置信度与下次验证：** medium；股价修复需半年报验证利润增长，同时关注大宗商品价格和社零数据。若半年报后全年盈利预期被上调，估值可迅速修复。

**Q5：Q5: 相比天味食品等同行，安井是否是更好的配置？**
- **为何关键：** 资本配置需考虑机会成本，确保标的在同类中具有最优风险收益比。
- **当前结论：** 安井在估值安全边际、现金流质量和新业务期权方面占优，天味在盈利能力和利润率上更突出。两者并非互斥，但安井当前估值折价25%（板块PE中位数25.3x vs 安井19x），且资产负债表更健康（净现金40亿，负债率19.88%），防御属性更强。对于稳健型账户，安井是更佳的底仓选择。
- **采用证据：** Peer comparison: 天味PE 20.1x, ROE 12.5% vs 安井PE 19.0x, ROE 9.6%，但安井OCF/净利润1.85x；安井股息率3.28% vs 天味3.88%（接近），但安井PB 1.81x更低；安井拥有清真/出口等期权价值，天味无类似增长点
- **最强反证：** 天味食品更高的ROE和毛利率意味着更强的内生增长和议价能力，其估值溢价合理；若市场偏好高盈利质量，资本可能更青睐天味。
- **模型/估值影响：** 维持安井19x PE估值中枢，同时没有因同行比较下调估值。在组合构建中，安井可作为防御成长型底仓。
- **置信度与下次验证：** medium；同行中报对比，验证现金流和利润质量差距是否收敛。

## Forecast Takeaways

### 预测结论

**1. 2026年净利润增速23.6%远超市场隐含的约12.5%，利润弹性来源于毛利率修复和经营杠杆，而非激进的收入假设。**
- 证据锚：2026Q1毛利率24.99%（2025年全年21.6%），Q1净利润5.63亿（+42.7% YoY），渠道反馈利润增速快于收入（KPE01）。
- 财务含义：若全年毛利率达23.2%，归母净利润可达16.8亿元，对应EPS 5.04元；若毛利率达24%，利润可超18亿。当前股价仅隐含15.3亿利润，存在明显低估。
- 置信度与风险：中等信心。关键风险在于下半年原材料价格反弹和餐饮需求失速，可能导致利润不及预期。

**2. 自由现金流和分红提供坚实的下行保护，FCF/市值比率有望从4.1%提升至5%以上，支持高股息和估值修复。**
- 证据锚：2026Q1 OCF/净利润1.85x，Capex 2025年实际8.73亿，预计2026年9亿，对应FCF约12亿（FCF/市值4.1%）。历史分红记录19年，2025年度合计每股4.32元（股息率约5%）。
- 财务含义：若盈利增长且资本开支受控，2027-2028年FCF可达15.5-19亿，FCF/市值将升至5-6.5%，为高分红和估值扩张提供基础。
- 置信度与风险：高信心。低负债和健康现金流使股息可持续，除非出现重大资本开支或盈利大幅下滑。

## Ranked Core Theses

### 论点1：盈利弹性被市场低估，当前估值未充分反映2026年净利润增长23.6%至16.8亿的确定性。

- **关键问题：** Q4: 当前PE定价是否充分？
- **核心证据：** 2026Q1毛利率24.99% (+1.66pp)，净利5.63亿 (+42.7%)；渠道反馈Q2利润增速快于收入 (KPE01)；成本端生猪/玉米/棕榈油期货下跌 (EV038-EV041)；分析师上调2026E净利至16.8亿 (Shared Model Update)。
- **最强反证：** Q1高增长有低基数效应，成本顺风若逆转可能使毛利率回落至22%以下，叠加消费疲软，利润可能仅15亿左右（熊方情景）。
- **财务传导：** 毛利率23.2% + 营收182.4亿 → 归母净利润16.8亿，EPS 5.04元。19x PE对应价值96元/股，较现价87.12元有10%上行空间；若市场给予20x PE，则值100元。
- **市场定价：** 当前市值290亿对应TTM利润15.3亿，隐含增长12.5%；我们的16.8亿对应2026E PE 17.3x，低于历史中位数。存在预期差。
- **证伪条件：** 半年报毛利率<22.5% 或 营收增速<10%，则盈利弹性证伪，下调至15亿利润。
- **当前裁决：** partial

### 论点2：强劲的资产负债表和高分红提供下行保护，赋予防御属性，且分红收益率具备吸引力。

- **关键问题：** Q4中的安全边际与分红部分
- **核心证据：** 资产负债率19.88%，货币资金49.8亿，短期借款8.9亿，净现金约40亿（约占市值13.8%）；2025年度累计每股分红4.32元，股息率约5%（Dividend context）；连续19年分红，经营现金流/净利润1.85x。
- **最强反证：** 若盈利下滑导致FCF不足以覆盖分红，或公司因资本开支需求削减分红，将削弱防御价值。但当前净现金和低CapEx需求使该风险较小。
- **财务传导：** 净现金和稳定分红为股价提供安全垫，同时也增强了在低利率环境下的资产配置吸引力。FCF/市值升至5%以上后，估值有修复动力。
- **市场定价：** 高分红率当前为3.3-5.0%，在消费板块中属于较高水平，但市场未给予估值溢价，反映了对业绩波动的担忧。
- **证伪条件：** 若公司取消或大幅削减分红，或未来CapEx激增至15亿/年导致FCF为负。
- **当前裁决：** partial

## Company Disaggregation

安井食品的经济单元可分解为三个核心部分：(1) 安井品牌速冻主业：公司绝对收入和利润来源，贡献约90%以上的毛利。产品涵盖火锅料（丸子、虾滑）、速冻面米（蒸煎饺、烧麦）、速冻菜肴（预制菜），通过经销商和特通渠道分销。该业务具有规模、品牌和渠道护城河，处于成熟但仍在品类扩张的阶段。2025年设计产能129万吨，2026Q1毛利率24.99%，渠道反馈Q2出货增10-15%。估值适用18-22x PE。 (2) 核心子公司：新宏业（小龙虾/水产）、新柳伍（鱼糜制品）、鼎味泰（高端预制菜）。这些业务体量较小但波动大，2026年新宏业恢复双位数增长，新柳伍利润承压，整体贡献有限，需观察其盈利稳定性和协同效果。可视为业绩弹性来源或风险点。 (3) 第二曲线期权：清真安斋项目（牛肉丸等）、出口东南亚。目前处于早期商业化阶段，8款产品动销良好，经销商网络初步搭建，但收入利润尚未形成规模。属于期权价值，不纳入核心盈利预测。以上分解基于2025年年报（EV030-EV035）及渠道调研（KPE01, KPE03），但子公司和新业务的精确财务数据仍缺失，需半年报补充。

## Autonomous Three-Year Forecast Model

基于运营驱动的三年独立预测，与共识无关。

## Thesis-to-Financial Bridge

1. **盈利弹性桥**：毛利率修复→毛利增加→利润弹性。
   - 公式：营业利润变动 ≈ 营收变动 × (毛利率 - 费用率) + 营收 × Δ毛利率。
   - 基线：毛利率23.2%（较2025年+1.6pp），营收182.4亿，毛利增加约7.34亿，扣除费用增长后归母净利增加约3.21亿至16.8亿。
   - 情景：牛方24%毛利率→利润18亿；熊方22%→利润15亿。

2. **现金流与分红桥**：FCF = OCF - Capex。2026E OCF 21亿（Q1年化调整），Capex 9亿→FCF 12亿。若盈利改善且资本开支受控，2027-28年FCF可升至15-19亿，为高分红和估值扩张提供基础。分红率按78%计，2026年DPS约3.93元，股息率4.5%。

3. **期权价值桥**：清真/出口业务尚处早期，给予20%概率实现30亿增量价值，计入期权桶6000万（概率加权），每股约1.8元。这是额外的想象力，不纳入核心估值。

风险：若成本反弹或需求恶化，以上桥接均面临压力。半年报是关键验证点。

## 公司运营方程
- **收入** = 出货量（万吨）× 综合出厂均价（元/吨）
- **毛利** = 收入 - 原材料成本（鱼糜/肉类/面粉/油脂等）- 包材 - 制造费用
- **营业利润** = 毛利 - 销售费用（约5.5-6%）- 管理费用（约2.0-2.5%）- 研发费用（0.5%）
- **归母净利润** = 营业利润 - 财务费用（接近零）- 所得税 - 少数股东损益
- **现金** = 经营现金流 = 净利润 + 折旧 - 营运资金变动；FCF = OCF - Capex
