## 一、公司快照与决策

瀚蓝环境（600323.SH）是中国领先的综合环境服务运营商，主营固废处理（垃圾焚烧发电）、能源供应（管道天然气/LNG贸易）和供水服务。2025年公司完成对粤丰环保的重大资产重组并表，收入与利润体量跃升，同时资本市场对其高负债、高应收、并购整合风险的担忧达到极致——PE被压缩至10.7倍（5年15.7%分位），股价较年内高点已回调约15%。

**经过多轮多空辩论，我们裁定多头的基本逻辑成立，但空头提出的风险绝不能被忽视。** 本备忘录确立了“增持”评级，但明确标注为“审慎/分阶段建仓”，对仓位上限（正常仓位的2/3以下）和推进节奏进行严格纪律约束，将8月半年报设为决定性验证节点。

## 二、商业模式与产业链定位

### 2.1 公司如何赚钱

瀚蓝环境的核心收入引擎是**生活垃圾焚烧发电**，其商业模式可拆解为：**垃圾处理量 × 处理费单价 + 上网电量 × 上网电价**。公司通过长期特许经营权协议锁定区域内的垃圾供应和处理费，具有显著的**区域垄断性**和**需求刚性**。

产业链定位清晰：
- **上游**：地方政府及其环卫部门（供应垃圾、支付处理费）、电网公司（购买电力）
- **中游**：公司自身——垃圾焚烧发电、能源供应、供水服务
- **下游**：居民、工商业用户

公司的核心护城河是**特许经营权壁垒**：固废处理、供水业务均需政府授权，新建产能进入同一区域极难。这是“类公用事业”估值的根基。

## 三、行业周期裁决与估值框架

### 3.1 周期定位

固废和水务板块已进入**“运营收获期”**。垃圾处理量稳定增长，资本开支高峰正在过去，工程业务收入持续下降（同比下降40.58%，系主动选择）。行业正从“建设期”切换到“运营期”，现金流生成能力将逐步释放。

能源业务受燃气价格波动影响，但公司以管网和LNG贸易为主，周期性弱于纯上游。

**整体判断：公用事业属性主导，下行周期防御能力强，弹性有限。**

### 3.2 估值桥梁与预期差

**当前PE 10.7倍（5年15.7%分位）到底定价了什么？**

市场正在用接近“破产清算”的视角给这家公司估值：
- 恐惧高负债（69%资产负债率，净债务约261亿）
- 恐惧高应收账款（203.57亿，占总资产32.7%）
- 恐惧粤丰环保整合风险与商誉减值
- 恐惧FCF永远为负

**我们看到了什么不同？**

回顾价格-EPS-PE分解：过去5年，公司EPS增长了83.2%，但PE被压缩了32.3%。**这是一家盈利持续增长，但估值被不断降级的公司。** 当前EPS代理值处于历史98.1%百分位，PE处于15.7%百分位——市场对优异的盈利增长给予极低的定价。

这种分裂正是预期差的来源：**市场将周期性的现金流低谷（并购整合期）误解为结构性的价值陷阱。** 一旦2026年半年报验证OCF改善和应收/营收比率下降，市场将被迫从“清算估值”切换至“分红估值”。

## 四、核心投研问题与多空辩论裁决

### 议题 G-1: FCF可持续性（核心变量）

| 维度 | 内容 |
| :--- | :--- |
| **初始怀疑** | 公司能否创造足以覆盖分红的真实自由现金流？ |
| **多头答案** | OCF 40.18亿，正确FCF=OCF-Capex=24.73亿，分红仅占34.6%，安全边际充足。资本开支高峰已过，FCF拐点可期 |
| **空头攻击** | Q1的OCF同比-18.45%；应收账款203亿堆积如山；信用减值损失激增近10倍（977万→9,379万）；OCF中有约12亿靠“占用上游供应商资金”支撑 |
| **PM裁决** | **多头胜（有条件）**。空头正确描绘了当下的现金流压力——Q1 OCF下滑和应收账款激增是真实信号，不可轻描淡写为“季节性波动”。但多头正确指出了周期规律：资本开支前瞻性投入完成后，现金回报期将开启。分红承诺（1.155元/股）以当前FCF计算覆盖倍数约2.6倍，安全边际真实存在但正在收窄 |
| **估值/仓位影响** | **核心不确定性，直接压制仓位上限和估值。** 仓位限制在正常仓位的2/3以下，不以证据不全的乐观假设建满仓 |
| **下一步验证** | **2026年半年报（8月）**：OCF是否环比改善？应收/营收比率是否拐点下降？这是本投资论点的“生死局” |

**关键数字复算：**
- 多头正确FCF = OCF 40.18亿 - 资本开支15.45亿 = **24.73亿**（利息支出已含在OCF中，不再重复扣减）
- 2025年实际分红 = 8.56亿（1.05元/股 × 约8.15亿股）
- 分红占FCF比例 = 34.6%
- 2026年承诺分红 = 约9.42亿（1.155元/股），占2025年FCF的38.1%
- **安全边际存在但收窄，空头对此的警惕是正当的**

### 议题 G-2: 公司质量 vs 便宜

| 维度 | 内容 |
| :--- | :--- |
| **初始怀疑** | 我们买了一家好公司，还是只因便宜而买入？ |
| **多头答案** | 毛利率从29%→38%，运营效率真实提升；特许经营权是真实的护城河；国资背景+确定分红记录 |
| **空头攻击** | 高负债、高商誉（13.93亿）、整合风险；三峰环境（601827）在PE更低（10.06x）、PB更低（1.01x）、毛利率更高（47.96%）、负债率更低（47.36%）的情况下，质量全面优于瀚蓝 |
| **PM裁决** | **空头在行业内相对比较上胜出**。在单一质量维度上，三峰环境确实是更安全、更便宜的同类替代品。但瀚蓝环境的分红承诺确定性（1.155元/股官方指引）和并购整合弹性（粤丰环保协同效应尚未释放）提供了更高的赔率 |
| **估值/仓位影响** | 瀚蓝环境不是行业内最优选，是**“中等质量、高赔率”**的绝对收益配置。组合构建中应将其定位为分红防御型持仓，而非高确信度的质量股 |
| **下一步验证** | 持续跟踪ROE、ROIC、负债率变化；比较瀚蓝与三峰后续季度的现金流与盈利修复弹性 |

### 议题 G-3: 证伪路径

双方的证伪路径均清晰且可量化：

- **多头逻辑被证伪的条件**（看空验证）：2026年半年报显示OCF同比继续下降、应收/营收比率继续恶化、公司削减或取消分红承诺。
- **空头逻辑被证伪的条件**（看多验证）：2026年半年报显示OCF/营收比率回升、应收/营收比率拐点下降、资本开支继续同比下降、分红承诺被重申或上调。

**风险管理可控——离场条件和加仓条件均已明确设定。**

- 出现不可预见的重大资产减值或法律诉讼

## 八、核心多空辩论与决策逻辑

### 最强的看多论据（为什么我们要参与）

1. **分红承诺是有法律约束力的官方指引**。2026年每股不低于1.155元的承诺，在交易所互动平台被管理层明确重申。这不是口头叙事，而是可执行、可验证的硬承诺
2. **FCF基本面真实存在**。正确计算的FCF（OCF-Capex）= 24.73亿，分红仅占34.6%。安全边际存在，即使是空头也承认“按保守计算分红占比50.6%”亦属健康范围
3. **估值极端便宜**。PE处于5年15.7%分位，EPS处于历史极高位——盈利增长未被市场定价
4. **共识偏见**：市场一致看空（技术破位+机构未抄底），逆向布局的赔率提高

### 最强的看空论据（为什么我们被限制仓位）

1. **Q1 OCF同比-18.45%是真实的现金恶化信号**。应被认真对待，不能被“季节性波动”解释消解。如果这个趋势延续，FCF将在2026年显著收缩
2. **应收账款203亿且信用减值损失激增近10倍**。粤丰环保的存量应收质量可能低于瀚蓝原有资产，存在未来减值扩大风险
3. **国补退坡风险未被市场充分定价**。年报明确提及此风险，若退坡15-20%，净利润损失1.5-2.5亿
4. **三峰环境是更安全、更便宜的替代品**。在行业层面，瀚蓝不是最优选

### 真正的分歧与核心判断

**分歧不在于“公司当前现金流好不好”——双方都同意当前FCF有压力。**

分歧在于：**这个现金流压力是周期性的（正在过去的并购代价）还是结构性的（商业模式固有缺陷）？**

我们的核心判断：**这是周期性的，但”周期“本身存在被证伪的风险。** 我们押注资本开支高峰已过、运营效率提升将逐步转化为现金流入，但这个押注需要8月半年报来验证。在此之前，用1/3仓位参与，以¥25.8元为硬底保护，是该判断下最优的风险收益配置。

### 仓位调整规则

| 触发条件 | 行动 |
| :--- | :--- |
| **半年报数据全面改善 + 分红上调** | 加仓至满仓（正常权重） |
| **半年报数据改善但分红未上调** | 加仓至2/3仓位，等待下一季度确认 |
| **半年报数据持平** | 维持1/3仓位，不下结论，等Q3验证/证伪 |
| **半年报数据恶化** | 立即清仓离场，不计成本 |
| **分红承诺被取消或修改** | 立即清仓离场，不论股价 |
| **股价有效跌破¥25元（基本面未改善）** | 执行止损，保护本金 |

## 十二、持仓与建仓结论

### 当前持有者

- **维持现有仓位，不建议在此价格继续加仓。**
- 将止损线设在¥25元（硬止损，对应股息率>4.6%），严格执行
- 密切关注8月半年报：若数据改善，可在确认后加仓；若恶化，果断离场

### 准备建仓者

- **立即以不超过目标仓位的1/3，在¥28元以下试探性建仓。**
- 最佳建仓区间：¥25.8-28.0元（对应4.12%-4.5%承诺股息率区间）
- 不要在半年报之前建满目标仓位——Q1 OCF下滑信号尚未被证伪或解释
- 8月半年报是决定性时刻，预留至少2/3仓位等待验证

### 防守性建仓锚

对于稳健型投资者，可在¥25.8元附近挂单等待（对应4.5%股息率），若市场因情绪进一步打压至此价位，是一个极佳的长线建仓点。但需确认分红承诺未被削减后方可执行。

---

**本次决策的核心纪律：** 我们承认多头的逻辑框架是正确的，但我们也必须诚实面对Q1数据发出的警告信号。这不是一个可以自信重仓的时点，但足够以受控仓位参与一个高赔率、有硬底保护的非对称机会。**8月半年报将决定这个论点是成立还是崩塌，在此之前，保持仓位纪律就是对风险最好的管理。**
