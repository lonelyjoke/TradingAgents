# 江西铜业 (600362.SH) 投资决策备忘录

## 公司快照

江西铜业是中国最大的综合性铜业集团，业务横跨铜矿开采、冶炼、加工及贸易。公司控股股东为江西铜业集团（持股45.72%），拥有国内领先的铜资源储量、全球前列的冶炼产能及庞大的有色金属贸易网络。公司产业链涵盖从上游矿山开采（铜精矿）到中游冶炼（阴极铜）、下游加工（铜杆、铜箔、铜管等）及贵金属（黄金、白银）、稀散金属回收的全流程。其盈利对铜价、矿产铜产量、冶炼加工费、贸易价差及库存价值变动高度敏感。当前总股本约34.63亿股，总市值约1,536亿元，2025年全年营收5,446亿元，归母净利润71.3亿元。

**评级：Underweight (减持)**

**一句话投资论点：** 这是一家资源禀赋雄厚、但盈利质量存硬伤的中国铜业龙头；当前繁荣顶点的悲观定价并不充分，市场为脆弱的毛利率、高企的存货与恶化的现金转化效率支付了过高的溢价。

## 商业模式与产业链价值分布

**如何赚钱**：
江西铜业的营收主要通过三大业务板块实现：1）**铜矿开采**（自产铜精矿及部分粗铜）；2）**冶炼与加工**（赚取铜精矿至阴极铜的加工费TC/RC，以及高端铜材产品的溢价）；3）**贸易**（大规模的有色金属及原材料贸易，赚取价差及物流服务费用）。利润的核心驱动是“矿山品位×采选成本”、“铜价×矿产铜权益产量”、“加工费×冶炼能力”、“贸易规模×价差”。

**产业链位置与风险映射**：
- **上游（矿端）**：江铜拥有国内最大的铜矿资源，矿山利润直接受益于高铜价，这是其核心价值所在。成本刚性（品位下降、人工、能源）是长期挑战。
- **中游（冶炼端）**：作为全球最大冶炼商之一，产能巨大，但冶炼加工费（TC/RC）极其低迷。年报称“普通铜材加工产能过剩，行业内卷加剧”。这使其在高铜价下，冶炼业务可能成为利润拖累，形成“半受益”结构。这是它与纯矿企的核心差异。
- **下游（贸易端）**：贸易业务贡献了80%以上的营收，但毛利率极低（<2%），其价值在于维系供应链关系、提供市场情报和套期保值规模，但对利润的实际贡献有限，且占用了天量的营运资本（存货、应收），大幅拉低了整体的现金转化效率。
- **同业对比**：上游矿企（如紫金矿业、西部矿业）能更直接地享受高铜价红利，盈利和现金质量更优；纯粹的中游加工企业在行业低谷期则更脆弱。江铜作为综合平台，其复杂的“利润-现金”错配正是市场博弈的焦点。

## 核心投资逻辑（投资主线）：繁荣顶点的交易价值

**核心判断（The Core Bet）**：我们赌的不是铜价崩盘，而是“繁荣溢价”的消退。当前19.2倍的PE，对应的是铜价高位运行、利润可持续增长、现金流健康、分拆期权实现的乐观预期。而现实的证据更指向一个繁荣顶点的组合：Q1亮丽利润的脆弱性、经营现金流的结构性压力、存货价值的高位风险，以及同板块内存在显著更优的配置选择。**市场的定价是“持续的繁荣”，而公司的证据是“脆弱的顶峰”。这个预期差是向下的。**

### 盈利与估值桥梁

**【盈利驱动因素映射】**
我们通过以下3-6个变量构建盈利桥梁：
1.  **SHFE铜价**：已验证（105,150元/吨）。处于历史高位，远期浅contango结构暗示市场不预期暴跌，但也未给出上涨溢价。**信号：中性偏强。**
2.  **矿产铜产量**：从年报信息推断，自产铜精矿产量稳中有升，但增速远不及冶炼和贸易规模的膨胀。**信号：微幅正贡献。**
3.  **综合毛利率**：Q1为5.07%（同比+0.78pp），但这主要受益于铜价上涨的行业性红利。年报对冶炼加工“内卷加剧”的描述表明，冶炼端利润可能持续承压。该指标的可持续性高度可疑，只要铜价横盘或回调，毛利率将迅速回落。**信号：短期改善，中长期脆弱。**
4.  **营运资本变动（存货与应收）**：这是令盈利“失真”的核心变量。666亿的存货在Q1铜价上涨时贡献了重估收益，同时也意味着巨大的下行风险。经营现金流的剧烈波动（从-69亿到+55亿）表明盈利尚未转化为可自由支配的现金。**信号：高风险、高波动，是盈利质量的负项。**
5.  **合同负债**：41亿，创历史新高。但正如辩论所揭，这更多可能反映的是铜价上涨后的名义金额增长，而非需求能见度的质变。其对未来收入的指引作用有限。**信号：偏正面但弱，待验证。**
6.  **分拆上市催化剂**：目前仍为传闻。**信号：无。属于未证实的期权价值。**

**【市场隐含预期分析】**
- **市场看似相信什么？** 市场给的19.2倍PE（TTM）和过去1年82.7%由估值扩张驱动的股价上涨，表明其相信江铜的盈利增长故事具有可持续性，甚至正在加速。这是一个“抱团确定性”或“交易繁荣”的定价。
- **我们同意与不同意什么？** 我们同意，铜的长期供需结构是有利的，江铜的资源规模是真实的。但**我们尖锐地反对市场为其支付的“品质溢价”**。这并非一家拥有高壁垒、高现金转化率的复合增长机器。它是一家受大宗商品周期和自身笨重业务模式双重束缚的、盈利质量偏低的龙头企业。市场为其支付了近20倍PE，但我们认为，基于其盈利质量，长期看它只值周期中值的12-15倍PE。
- **什么能使市场重新定价？** Q2或Q3一份惨淡的财报（毛利率骤降、现金流再次转负、合同负债萎缩），或者铜价一次超过15%的回调，都将迫使市场重新审视其盈利的脆弱性，从而引发“杀估值”过程。届时PE可能迅速向15倍甚至更低靠拢。

### 牛市/熊市情景与决策逻辑

**【牛市情景（概率：20%）——被证实的盈利加速】**
- **条件**：SHFE铜价持续上行突破12万；Q2、Q3毛利率稳定在5.0%以上；经营现金流连续多个季度为正且与净利润匹配；合同负债继续创新高，证明真实需求推动；分拆上市获得实质性进展。
- **结果**：全年利润可达130亿元以上，PE（基于全年利润）迅速被消化至12倍左右，股价有望重回55-60元甚至更高。届时应全面转为看多。
- **我们的看法**：这是一个低概率但高赔率的情景，但完全依赖多个目前尚不成立的条件同时完美兑现。

**【基础情景（概率：35%）——高位震荡的牛熊胶着】**
- **条件**：SHFE铜价在10万附近高位震荡；Q2/Q3毛利率回落至2025年水平（4.4-4.8%）；经营现金流季节性波动，全年勉强打平或微正；合同负债高位钝化；分拆上市悬而未决。
- **结果**：全年利润约80-100亿元，当前股价对应的PE在15-19倍之间波动。估值缺乏扩张动力，股价大概率随铜价和情绪在35-45元箱体震荡。
- **我们的看法**：这是最可能出现的情况。在此情景下，江铜没有超额收益，是一个平庸的持有标的，远不如布局品质更优的同类资产。

**【熊市情景（概率：45%）——繁荣顶点的风险释放】**
- **条件**：铜价因宏观扰动或供给释放跌破9.5万（甚至9万）；Q2合同负债和毛利率同步回落，证实Q1只是“虚火”；经营现金流重新恶化；分拆上市停滞。
- **结果**：盈利预期回落至55-70亿，同时触发对666亿存货的减值担忧。PE将被动抬高或股价下跌挤压估值泡沫。股价将向35元以下，甚至30元的安全锚点回归。
- **我们的看法**：此情景概率不低且赔率负面（下行空间15-20%）。当前脆弱的基本面和技术面组合，使得该路径的风险收益比最差。这构成了我们减持建议的核心逻辑。

**决策逻辑**：综合评估，熊市情景的概率（45%）远高于牛市情景（20%），且熊市情景的下行赔率与牛市情景的上行赔率相当。当前的价格并未给应对熊市情景的风险提供任何补偿。因此，加权期望值为负。Underweight是最能反映该风险收益关系的合理决策。

## 营收与利润增长可持续性 / 放量条件

盈利（归母净利润）增长的可持续性，取决于以下变量的共同作用：

| 变量 | 当前状态 | 可持续性评估 | 验证来源 / 证伪信号 |
| :--- | :--- | :--- | :--- |
| **1. 铜价中枢** | 10.5万/吨 (Q1均价) | **不可持续上移**：远期浅contango不支持持续大涨，上行空间有限，维持高位即是最好情况。 | **证伪**：若SHFE铜跌破9.5万/吨，价格驱动的利润将消失。 |
| **2. 合同负债转化** | 41亿 (同比+21%) | **待验证，持续性弱**：增速与铜价涨幅趋同，名义效应可能大于真实需求结构性改善。 | **验证**：Q2若在铜价持平时仍能环比持平或增长。**证伪**：Q2回落至35亿以下。 |
| **3. 毛利率趋势** | 5.07% (同比+0.78pp) | **难以持续**：前提是铜价不跌且冶炼加工费/成本不恶化。历史上毛利率围绕4.5%中枢波动。 | **证伪**：Q2毛利率回落至4.4-4.5%区间，证明Q1改善更多是价格的一次性红利。 |
| **4. 经营现金流质量** | Q1为+55亿 | **单一季度无法证明**：2025年全年为-69亿，Q1的好转需后续季度验证。季节性波动巨大。 | **验证**：连续两个季度（Q2、Q3）经营现金流为正且强劲。证伪：Q2再度转负或大缩水。 |
| **5. 存货减值风险** | 666亿存货 | **风险敞口持续存在**：只要铜价处在历史高位，下行时的减值风险就与日俱增，会一次性吞噬利润。 | **证伪**：铜价稳定在10万以上超过两个季度，或公司显著降低存货。 |

**结论**：上述五个变量中，无一能被确认为可持续的向上趋势。铜价仅是高位维持，合同负债和毛利率的改善证据不充分，现金流的改善仍待确认，而存货风险敞口巨大。因此，我们判断当前的增长不具备线性外推的基础。需要半年报甚至三季报的连续数据来证明是趋势性反转，届时再评估转变评级。

## 相对走势与指数联动

- **基准与行业篮子**：由于中证500风格基准数据缺失，我们主要参照由8只铜行业股票构成的等权重行业篮子。
- **多窗口表现**：在**250天窗口**，江铜相对行业篮子跑赢33.33%，Beta为1.12，相关性高（0.82）。这反映了过去一年铜价牛市周期中，江铜作为龙头的累计超额收益。但在**60天窗口**，情况发生转变：江铜回报率为-14.31%，而行业篮子仅-8.82%，**跑输5.49%**，且相关性高达0.85。近期**20天窗口**内，两者表现趋于一致（-9.02% vs -9.25%），显示近期下跌是板块性共振。
- **核心结论：Alpha退潮，Beta跟随**。江铜的长期超额正在被近期的持续跑输所侵蚀。这并非简单的板块回调，而是市场在铜价仍处高位时，对江铜自身盈利质量和表达效果进行重新评估和修正。股价已经从过去的“领涨”变为在反弹中弹性较弱、在下跌中紧跟的“Beta跟随者”。对已持有者而言，这意味着寄望于它能独立走出Alpha行情的概率在降低。对建仓者，该品种在板块内部的相对吸引力正在下降。

## 核心风险

- **铜价大幅回调（跌破9.5万/吨）**：对666亿元存货带来巨大减值压力，足以吞噬全年利润，是最大下行风险。
- **盈利质量“脱轨”**：Q2或Q3财报中，毛利率回落至4.5%以下的同时经营现金流再度转负，将证实Q1的改善不可持续，引发“戴维斯双杀”。
- **同板块替代效应**：鉴于西部矿业等在盈利质量、增长和估值上的明显优势，看好铜行业的资金可能持续从江铜流出，转向更优标的，导致其Beta跟随但缺乏Alpha。
- **“分拆预期”落空**：若长时间停留在公告阶段无实质进展，或最终分拆的是低估值加工业务，将使得部分基于此乐观预期的筹码出清。
