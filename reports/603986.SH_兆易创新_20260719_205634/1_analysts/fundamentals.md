## 一、公司如何赚钱：商业模式解构

兆易创新是一家**无晶圆厂（Fabless）集成电路设计公司**，核心业务为**存储器（NOR Flash、SLC NAND、利基型DRAM）、微控制器（MCU）、传感器和模拟芯片**的研发与销售。公司不拥有晶圆制造产能，晶圆代工外包给中芯国际、华虹等代工厂，封装测试由第三方完成。

**收入驱动公式**：
```
总收入 = Σ(各产品线出货量 × 平均售价)
```
其中存储产品的价格弹性极大，是利润波动的核心变量。

**利润传导链**：
```
出货量 × ASP → 收入 → 减晶圆/封测成本 → 毛利率 → 减研发费用（~17%收入）→ 减销售管理费用 → 营业利润 → ±非经常性损益 → 净利润
```

**关键业务单元**（据2024年半年报及2025年年报披露）：

| 业务线 | 披露程度 | 经济角色 | 周期属性 |
|--------|---------|---------|---------|
| 存储器（NOR/SLC NAND/DRAM） | 未拆分收入 | 最大利润池，当前占收入估计>70% | 强周期，当前处于超级涨价期 |
| MCU微控制器 | 未拆分收入 | 第二增长曲线，物联网/汽车电子 | 中周期，2025年触底回升 |
| 传感器 | 未拆分收入 | 辅助业务 | 温和周期 |
| 模拟芯片 | 未拆分收入 | 种子业务，2025年新进入 | 早期，尚未贡献利润 |

**关键事实**：公司**未在年报中拆分各产品线的收入、成本和毛利率**，这对精确建模构成重大障碍。拆分数据需要在2026年半年报发布后补充。

## 二、当前利润爆发的核心驱动：利基存储超级周期

### 2.1 供需双轮驱动的涨价逻辑

**需求侧**：AI服务器、数据中心对高容量存储的需求激增；端侧AI（手机、PC、IoT）推动存储容量升级；汽车智能化增加NOR Flash/DRAM搭载量。

**供给侧**：海外存储原厂（三星、SK海力士、美光）持续收缩利基型存储（SLC NAND、NOR Flash）产能，将晶圆产能转向HBM、DDR5等高附加值产品。这造成了**结构性供给缺口**。

**证据链**：
- TrendForce预测：2026年下半年SLC NAND价格将**上涨120-170%**（KPE03，行业调研引用）
- KeyBanc亚洲供应链调研后大幅上调DRAM/NAND价格预测，认为供给紧张延续至2027年（KPE01）
- 公司2026年H1业绩预告：收入115亿（已超2025全年92亿），归母净利润69亿

### 2.2 量价齐升的财务传导

### 2.3 季度节奏拆解

从H1预告反推Q2单季：
- **Q2收入**: 115亿 - 41.9亿 = **~73.1亿**（环比+74.5%）

- **Q2扣非净利润**: 48.5亿 - Q1扣非（未披露，但Q1净利14.6亿中扣非占比估计很高）≈ 推测Q2扣非~34亿

Q2净利润（54.4亿）远超Q1（14.6亿），说明**涨价效应在Q2集中释放**。收入环比增长74.5%而净利润环比增长272%，反映了极高经营杠杆：收入增长的大部分转化为利润。

### 6.3 核心押注与证伪条件

**核心押注（Core Bet）**：SLC NAND存储超级周期可持续至2027年，供给缺口（海外产能退出）叠加AI需求结构性增长，使公司在本轮周期中实现远高于历史均值的盈利水平。

**多头必须证明**：
1. H2 2026 SLC NAND价格继续上涨（TrendForce预测120-170%）
2. 海外原厂（美光、三星）没有快速回归利基市场
3. 公司出货量保持增长，客户库存健康
4. 扣非净利润在2026H1的48.5亿基础上，H2保持或温和下降

**空头必须证明**：
1. 存储价格已在2026Q2见顶，H2开始回落
2. 海外产能2027年回归导致供给过剩
3. 公司H1利润中含大量不可持续的非经常性收益
4. 当前PB 96.5%百分位已充分反映周期顶部

1. 2026Q3收入环比下降超过10%（需求放缓）
