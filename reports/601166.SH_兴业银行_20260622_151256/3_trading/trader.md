# 最终裁决：601166.SH 兴业银行 — 研究经理备忘录

**日期：** 2026-06-22  
**评级：** **Underweight（低配）**  
**行动：** 现有持仓逐步减少至战术性底仓；新资金暂不入场  
**前期评级：** 本轮为独立裁决  

---

## 一、核心投资论点

**核心投资假设：** 兴业银行是否是一个被错误定价的价值股，即将因NIM触底和PB均值回归而修复——还是一个存在结构性资本陷阱、管理层不作为、股东回报不可持续的"价值陷阱"？

**关键验证点：** 
- 管理层行为（回购、分红）是终极诚实信号
- 可转债转股停滞揭示资本约束
- NIM下行是否见底

**预期差判断：** 市场并非过度悲观，而是合理前瞻性定价了ROE结构性下行。0.44x PB不是错误定价，而是对"低增长+资本约束+管理层不作为"的合理折价。多头期待的均值回归缺乏催化剂和硬证据支撑。

## 二、关键证据链

### 2.1 管理层信号：致命硬伤

| 证据项 | 来源 | 内容 | 置信度 |
| ------ | ---- | ---- | ------ |
| 拒绝回购 | 2026-06-04上证e互动 | "目前，我行暂无回购计划。" 股价PB 0.47x，净资产39.34元 | ***核实*** |
| 实质削减末期分红 | 2025年年报/2026年一季报 | 末期分红0.501元 vs 上年1.06元（-52.7%） | ***核实*** |
| 市值管理叙事化 | 同上 | "制定《市值管理制度》...提质增效重回报" 缺乏量化承诺 | ***核实*** |

**分析：** 在股价深度破净时拒绝回购，是最诚实的内部信号——管理层对自身资产价值缺乏信心。末期分红大幅削减，叠加中期0.565元后全年仅持平，反映：
- 利润压力（2026Q1归母净利润同比-5.89%）
- 内生资本补充能力不足（可转债转股停滞）
- 股东回报意愿弱化

### 2.2 资本陷阱：被验证的熊方核心论据

| 证据项 | 来源 | 内容 | 置信度 |
| ------ | ---- | ---- | ------ |
| 可转债转股停滞 | 上证e互动 | 转股价20.63元 vs 股价~17.32元，差价持续60-75日 | ***核实*** |
| 核心一级资本充足率趋势 | 2025年年报 | 从2021年9.81%降至2025年9.30%（-51BP） | ***核实*** |
| 内生资本消耗 | 2025年年报经营分析 | "内生性资本补充能力不足仍是未来发展面临的挑战" | ***核实*** |

**分析：** 可转债413.5亿元，2027年12月到期，距到期约1.5年。若不转股需清偿，将严重透支资本。这构成一个**结构性两难**：
- 促进转股需要股价>20.63元，但当前市净率0.44x，利润增速为负
- 维持股东回报与保留资本不可兼得

资本充足率下行趋势与可转债困境形成负反馈循环，这与2020-2021年的情形本质不同。

### 2.3 财务数据：静态平稳，动态隐忧

| 指标 | 2025全年 | 2026Q1 | 变化方向 | 置信度 |
| ---- | -------- | ------ | -------- | ------ |
| 净息差 | 1.57% | 1.43% | 再降14BP | ***核实*** |
| 不良率 | 1.09% | 1.09% | 持平 | ***核实*** |
| 拨备覆盖率 | 237.67% | 235% | 降2.67pct | ***核实*** |
| 归母净利润增速 | +0.68% | -5.89% | 转负 | ***核实*** |
| 利息净收入增速 | -1.19% | -0.47% | 收窄但仍负 | ***核实*** |

**分析：** 资产质量"静态平稳、动态不明"：
- 不良率1.09%稳定，但地产展期、城投化债的结构性风险被时间线延长
- 拨备覆盖率从高位回落（238%→235%），利润缓冲垫在变薄
- 在未提取到"逾期贷款/关注类贷款迁徙率"或"不良生成率"季度数据的情况下，无法判断资产质量的真实趋势

**关键数据缺口：** 不良生成率、逾期贷款率、关注类贷款迁徙率。这些是未来两个季报（2026年半年报、三季度报）的核心观察项。在当前缺失状态下，**对资产质量只能持中性假设**——既不假设改善也不假设恶化，但认可时间延长会稀释拨备覆盖率并提供缓冲。

### 2.4 手续费与新兴业务：差异化但体量不足

| 债券做市 | "人民币债券交易量近7万亿元" | 未单独披露 | ***核实*** |

**分析：** 托管和做市构成一定轻资本收入，但2.24%的利润贡献不足以对冲息差收窄的负面影响（NIM从1.57%再降14BP至1.43%）。综合化经营"十五五规划"需要3-5年才能贡献显著利润。

## 三、估值与概率收益结构

### 3.1 当前估值

| 指标 | 数值 | 来源/分析 |
| ---- | ---- | --------- |
| PB | 0.44x | Tushare daily_basic，2026-06-19最新 |
| PE_TTM | ~5.06x | 基于2025年归母净利润228.91亿元、股价17.32元 |
| 股息率TTM | 6.15% | 全年合计约1.066元/17.32元 |

**周期-估值校准：**

| 维度 | 现状 |
| ---- | ---- |
| 估值水平 | 极低（PB 0.44x） |
| 盈利能力 | 偏低且下行（ROE 8.69%→8.14%→实际值更低，NIM持续压缩） |
| 归类 | **低估值 / 低繁荣** |

**分析：** 
- 低PB反映了市场对低ROE和结构性下行风险的折价
- 6.15%股息率高但**稳定性不通过**，末期分红实质削减削弱了"防御性股息"标签
- 低Prosperity/低Valuation组合的核心问题：悲观是否已充分定价？目前的答案是否定的——0.44x PB看似极低，但对应8-9%的ROE仅给0.4-0.5x PB是合理折价，并非极端错误定价

### 3.2 胜率/赔率评估

| 评估项 | 判断 |
| ------ | ---- |
| 上行概率 | ≤35% |
| 潜在上行空间 | 0.55x-0.60x PB（对应股价~21-23元，+20-30%），前提：NIM见底+恢复分红+管理层改变 |
| 下行概率 | 30-40% |
| 潜在下行空间 | 0.36-0.40x PB（对应股价14-15.5元，-10-18%），路径：NIM再降10-15BP+资产质量暴露+分红削减 |
| 高概率低偏离 | 25-35%概率：股价在15-19元区间窄幅震荡 |

**概率/收益比不佳的核心原因：** 上行需要多重同时兑现（NIM触底+管理层改变+经济复苏），且每个条件目前均无证据；下行路径清晰且可由单一事件触发。

### 3.3 催化剂与证伪信号

| 催化剂/观察项 | 时间/数据来源 | 对判断的影响 |
| ------------- | ------------ | ------------ |

**证伪信号清单（何时从Underweight转为Sell）：**

2. 全年分红总额较1.066元再减10%以上
3. 被迫进行低转股价的债转股或低价增发融资
4. 核心一级资本充足率降至9.0%以下

## 八、最终裁决总结

**评级：Underweight（低配）**

兴业银行表面上具备极低估值（0.44x PB, 6.15%股息率）的吸引力，但本轮分析暴露出三个致命伤：

1. **管理层信号硬伤**：在股价深度破净时公开拒绝回购，同时实质性削减末期分红（0.501元 vs 上年1.06元），这是比任何外部数据都更可信的内部悲观信号。

2. **结构性资本陷阱**：可转债转股停滞（转股价20.63元 vs 股价~17.32元）、核心一级资本充足率趋势下行、分红与内生增长不可兼得，构成负反馈循环。

3. **相对弱势确认**：股价1年跑输申万银行指数23.5%，2026年6月18日量价齐跌，机构止亏盘行为不能简单解读为"空头衰竭"。

**当前0.44x PB不是市场错误定价，而是市场对"低增长+资本约束+管理层不作为"的合理折价。多头的均值回归叙事缺乏催化剂和硬证据支撑，胜率/赔率结构不支撑现在建仓。**

**核心行动：**
- 现有持仓：逐步减仓至战术性底仓（≤银行股标准配置的50%）
- 新资金：暂不入场
- 观察条件：管理层回购公告、NIM连续两季降幅收窄、分红政策逆转
