# 投资组合经理决策备忘录：000967.SZ 盈峰环境

**公司概览**

盈峰环境（000967.SZ）是中国环卫装备与城市服务领域的头部企业，2025年营收约138.4亿元。公司业务分三大板块：智能装备（环卫车辆及设备，收入占比约42.7%）、智慧服务（城市环卫运营，收入占比约45.7%）、智云计算（含算力租赁等，收入占比约1.8%）。公司控股股东为宁波盈峰资产管理有限公司，实际控制人为何剑锋，中联重科为第二大股东（12.6%）。2025年公司新增海外订单5.4亿元（+445%），同时将“智云计算+算力租赁”列为战略方向，引发市场对其“AI+环卫”转型的定价重估。当前股价10.6元，对应59.7倍PE、1.95倍PB，总市值约348亿元。

---

## 最终评级：卖出

**核心投资逻辑（一句话）：** 市场以59.7倍PE定价的是一个“盈利质量在通过资产负债表渠道系统性恶化、而转型故事尚未进入可验证商业化阶段”的公司——这个定价组合从概率/赔率看已极度不利。

**已持有者与准备建仓者的行动指引**

**对已持有者**：当前的核心任务是**清仓或大幅减仓**。不保留核心配置仓位。若组合有最低配置要求，可将仓位压缩至1%以下并明确标注为“风险暴露观察仓”。减仓窗口应在2026年6月中旬前完成，避免半年报（8月）披露前流动性进一步收缩。卖出优先于等待“反弹回本”——60倍PE的股票在资产质量恶化周期中，反弹往往以更急的跌势告终。

**对准备建仓者**：当前价格和基本面条件下，**不应建立任何新仓位**。转型故事（海外订单、合同负债环比增长）是真实的线索，但已被高估值充分甚至过度定价；而资产负债表恶化的核心风险——长期应收款暴增、信用减值翻倍、借债分红——尚未被市场开始定价。在半年报验证前，不存在“以合理价格买入转型期权”的安全条件。等待以下明确的入场信号出现后，再以5-7.5元（对应核心业务合理PE 15-20倍）为基准考虑小规模验证仓。

---

## 安全价格区间 / 防御性建仓锚

**结论：在当前财务结构快速恶化的背景下，无法设定可靠的防御性建仓锚。**

任何基于稳定ROE、可持续分红或低周期盈利的估值模型，在面对2026年Q1长期应收款单季暴增17.2亿、经营现金流-12.3亿、资产负债率一年半内从39.4%飙升至53.6%的财务现实时，其基础假设均已失效。为一家正通过大规模向客户垫资来维持增长的公司设定“安全价格”，是不负责任的。

当公司的商业模式从“提供装备+服务”演变为“为客户提供融资”时，其估值逻辑应从PE/PB转向PB/资产质量。当前1.95倍PB，在信用成本上升和潜在坏账风险面前，不构成任何安全边际。

**只有在以下条件至少两条同时满足后，才可重新评估防御性建仓锚：**
1. 长期应收款停止增长并开始回落至18亿以下（证明垫资型增长已被遏止）
2. 连续两个季度经营性现金流显著为正，且能覆盖投资性现金流出
3. 信用减值损失增速放缓或转负，应收账款/收入比率持续下降

届时，合理估值锚可设在核心业务PE 15-20倍，对应股价约5.0-7.5元（基于2025年正常化利润约5.5亿、每股净资产约4.5元）。该区间反映的是环卫装备+服务主业的价值底线，算力租赁等新业务仅作为低概率期权，不纳入安全锚计算。当前价格10.6元距此区间还有29%-53%的下行空间。

## 商业模式与产业链定位

盈峰环境的商业模式可理解为“环保装备制造+城市服务运营”的双轮驱动，并新增了“数智化”（算力租赁）作为第三极。

**产业链位置：**
- **上游**：环卫装备的上游是商用车底盘（与卡车/专用车产业链重合）、钢材、电池（新能源环卫车）；环卫服务的上游主要是人力成本和燃油/电力成本。公司不掌握上游核心环节，成本控制依赖于规模采购和车辆新能源化。
- **公司所处位置**：中游装备制造商与下游服务运营商一体化。装备销售给政府环卫部门或环卫服务公司；服务直接与地方政府签订3-5年运营合同，负责清扫、转运、垃圾收运等。
- **下游**：终端需求来自地方政府环卫预算，具有刚性但价格敏感、回款周期长（政府客户特点）。海外市场正在拓展中，客户结构和回款模式尚未系统披露。

**收入与现金流的核心变量：**
- **装备端**：销量、单台售价、新能源化比例、毛利率（竞争激烈，约27%）、应收账款/收入比（垫资强度）。
- **服务端**：在手合同额、年化合同额、续约率、日服务价格、人力/油耗成本、政府回款节奏（决定经营现金流）、合同负债（预收款，反映订单池）。
- **算力租赁**：机柜规模、GPU型号/数量、利用率、租赁价格、毛利率（当前17.9%）、资本开支（采购GPU/数据中心）。

**产业链位置对估值的影响：**
装备+服务是低增速、高垫资、与地方政府财政紧耦合的生意，合理PE应在15-20倍。算力和低空经济目前是“未来期权”，其估值应独立评估且因关键商业变量缺失而不能进入核心价值。市场以60倍PE整体定价，是将“高弹性期权”的想象空间错误地灌注到了“重资产+高垫资”的主业上。

**相关上市公司（未经验证的产业链参考）：**
上游：商用车底盘（东风汽车等）；中游环卫装备同类公司（无直接A股同等规模竞争者）；下游：城发环境（垃圾焚烧，不同细分）、中原环保（水务，不同细分）。

## 营收与利润增长可持续性 / 放量条件

2025年公司归母净利润5.5亿，2026年Q1利润2.14亿（季节性模式，不应简单年化）。市场定价隐含的增长假设远超现有证据支撑。决定增长能否持续或放量的核心变量如下：

| 增长驱动 | 当前证据 | 可验证/可推断/缺失 | 对估值的影响 |
| :--- | :--- | :--- | :--- |
| 海外订单转化 | 2025年新增订单5.4亿（+445%），但交付节奏、毛利率、回款模式未披露 | 订单已验证，收入转化和利润质量缺失 | 即使全额交付、毛利率额外高10个点，增量税后利润仅约4000万级别，对5.5亿利润池增厚不到8% |
| 环卫服务合同负债 | 2026Q1合同负债4.81亿，同比零增长，环比+21% | 环比增长已确认，同比零增长揭示非加速扩张 | 合同负债同比停滞，结合长期应收款暴增，说明订单增长依靠的是垫资而非需求有机爆发 |
| 核心装备收入增速 | 2025年装备收入59.1亿，增速14%，毛利率下降趋势中 | 年度增速已放缓，需更细分的月度/季度数据 | 装备是利润主体，14%增速只能支撑15-20倍PE，不可能支撑60倍 |
| 算力租赁放量 | 2025年收入2.48亿（毛利率17.9%），管理层定性为“暂未形成重大贡献” | 营收确认但体量极小，GPU型号/数量/利用率/客户均缺失 | 即使翻倍至5亿，增量净利仅2000-3000万级别，只能将60倍PE拉至57倍，无法改变估值结论 |
| 资本开支与融资约束 | 2025年购建固定资产10亿已将经营现金流全部吃掉；2026年1月公告拟注册超短融和中期票据 | 资金紧张已确认 | 算力集群、装备升级、海外扩张三驾马车同时运转，需要远超10亿的资本投入，要么削减分红、要么继续加杠杆——两者都将伤害估值 |

**增长的硬约束：**
2025年全年经营现金流9.63亿，购建固定资产花了9.97亿。2026年Q1经营现金流进一步恶化至-12.3亿，长期应收款单季暴增17.2亿。公司正在以“垫资换订单+借债维持投资”的模式运行。这种增长在财务上不可持续——当资金链或客户信用风险任何一个环节出现问题，增长将戛然而止。2026年半年报将是检验这个约束是否爆发的关键窗口。

## 核心投资论点

### 1. 预期差：市场定价了什么 vs. 我们看到了什么

**我们认为市场当前（10.6元/59.7倍PE）定价的隐含假设是：**
- 海外订单将持续高增并顺利转化为利润（年化20%+的EPS增长）
- 合同负债增长代表需求能见度持续改善
- 算力租赁具备大规模商业化前景，是“AI+环卫”的稀缺平台
- 经营现金流的恶化是季节性的、可控的经营波动

**我们从财务报告中读到的现实是：**
- **长期应收款**：2025年末13.72亿 → 2026年Q1末30.96亿，单季度暴增17.24亿。这个增量是2025年全年净利润（5.5亿）的3.1倍，是全年经营现金流（9.63亿）的1.79倍。这说明公司通过大规模融资垫资换取订单，而非依靠产品竞争力。
- **信用减值损失**：2026年Q1同比+104.84%，增速远超收入增速。应收账款/收入比率仍高达39%，客户信用风险正在积聚。
- **财务杠杆**：资产负债率从39.4%（2024年末）升至53.6%（2026年Q1末），一年半内攀升14.2个百分点。有息负债翻倍，同时维持现金分红——“借债分红”是A股中罕见的危险信号。
- **合同负债同比零增长**：2026年Q1合同负债4.81亿，与2025年Q1完全相同。需求能见度未在同比层面改善。

**预期差的核心：** 市场仍在为“转型故事”支付60倍PE的高溢价，而报表端揭示的现实是：盈利质量在恶化、资产负债表在变脆弱、增长需要通过大规模垫资来维持。当半年报继续印证这个判断时，估值体系将从“PE视角”（看利润表）向“PB/资产质量视角”（看资产负债表）切换——届时，PE倍数将不再是估值锚，PB将从当前1.95倍进一步压缩。

### 2. 估值桥接：当前价格需要什么才能成立？

**方式一：利润增长填平高PE（不可能路径）**
- 59.7倍PE对应隐含年化EPS增速要求：约25%-30%才能将前瞻PE降至30倍（仍高于行业可比）。
- 当前盈利桥接：海外增量约4000万利润 + 算力乐观增量约3000万 + 核心业务5%增长约2750万 = 合计约1亿利润增量，总利润约6.3亿，前瞻PE约55倍。
- 结论：现有证据支撑的最大乐观情景下，PE仍高达55倍，根本填不平。

**方式二：PE倍数向合理区间收缩（大概率路径）**
- 核心业务（装备+服务）合理PE 15-20倍，现金流质量需打折扣，给予18倍 = 99亿市值。
- 算力租赁：营收2.48亿×乐观1.5倍PS=3.7亿市值（已极慷慨）。
- 海外扩张期权：增量5000万利润×20倍PE=10亿市值（仅当交付和回款被验证后才成立）。
- 总和：112.7亿，对应股价约3.4元/股，较当前10.6元有68%下调空间。
- 此场景是在“不出现系统坏账”假设下的保守/中性估值。若坏账爆发，估值将进一步向PB 0.8-1.0倍（对应3.6-4.5元）靠拢。

### 3. 核心判断：押注资产质量恶化被重新定价

**核心赌注：** 市场尚未开始为长期应收款暴增、信用减值翻倍、负债率跳升和借债分红等资产质量恶化信号定价。这些信号是量化、即时、已发生的——而市场仍然沉浸在“AI+环卫”的转型故事中。当2026年半年报披露继续印证这些趋势时，预期差将迅速闭合。

**这个判断与多头的根本分歧：**
- 多头认为：海外订单+合同负债环比是增长加速的前瞻指标。
- 我们认为：长期应收款暴增揭示的是“增长的代价”——通过垫资换取订单，合同负债并未同步增加（同比零增长），说明这是信用扩张而非需求扩张。高质量的增长应伴随合同负债的同步上升和经营现金流的同步改善，三者同时发生的证据不存在。

## 辩论裁决：长期应收款——刺穿多头叙事的利剑

多空辩论的转折点出现在空头对资产负债表的深层挖掘。多头反复强调“合同负债环比+21%”“2025年经营现金流9.63亿”“海外订单445%增速”三个证据点。但空头没有停留在利润表，而是直指资产负债表断层：**长期应收款2025年末13.72亿→2026年Q1末30.96亿，单季度增加17.2亿。**

这个数字直接回答了“钱去了哪里”的灵魂拷问。2025年全年经营现金流9.63亿和账面利润5.5亿，在单季度17.2亿客户融资面前微不足道。空头由此论证：合同负债的增长不是因为产品竞争力强，而是因为公司通过激进的融资垫资换取了订单。

配套佐证——信用减值损失同比翻倍、资产负债率从39.4%飙升至53.6%、以及A股罕见的“借债分红”模式——构成了一幅“增长质量恶化”的图景。多头的“戴维斯双击”叙事在这些硬证据面前失去了根基。

**多头回击中未解决的致命矛盾：**
- 多头称经营现金流仅是“季节性垫资”，但无法解释为什么2026年Q1长期应收款的单季增量（17.2亿）远超历史上任何一季的正常营运资金波动。
- 多头称合同负债环比+21%是需求信号，但无法解释为什么同比零增长——如果2025年Q1已可做到4.81亿预收款，2026年在“新增海外订单”和“服务合同扩张”下为什么完全持平？
- 多头引用67亿货币资金证明融资能力强，但回避了总负债同期增加62亿、有息负债翻倍的事实。

**研究经理的裁决已被充分验证：空头胜利。** 攻击点从“PE太贵”（容易被故事反驳）升级为“资产负债表恶化”（故事无法覆盖），逻辑上无懈可击。

## 关键证据审查与数据校验

在最终决策前，对几个争议数据进行校验：

1. **合同负债同比零增长的确认：**
   - 2025年Q1末合同负债：约4.81亿（来自辩论记录中卖方引用）
   - 2026年Q1末合同负债：4.81亿（来自Q1季报/催化剂审核）
   - **结论：** 同比确为零增长。多头反复强调的“环比+21%”掩盖了同比停滞这一更重要的事实。

2. **长期应收款暴增的确认：**
   - 2025年末：13.72亿（来自年报名目/财务报告情报）
   - 2026年Q1末：30.96亿（来自财务报告情报的`long_term_equity_investments`行和辩论中的反复验证）
   - **结论：** 单季度+125.6%，金额+17.24亿。此数字决定性地大，且与经营现金流的恶化（2026年Q1 -12.3亿）相互印证。

3. **经营现金流的年度vs季度矛盾：**
   - 2025全年OCF：9.63亿（良好）
   - 2026年Q1 OCF：-12.32亿（极差）
   - **结论：** 不能因年度数据良好而忽视季度恶化的强度。Q1的异常流出需要结合长期应收款暴增来理解——垫资正在加速。

4. **借债分红的确认：**
   - 2025年现金分红：约4.26亿
   - 有息负债同期：从约21亿升至约46亿
   - 2026年1月公告：拟注册超短融和中期票据
   - **结论：** 公司确实在以新增负债支持分红，同时维持高额垫资。此行为损害财务稳健性，且对股东回报的真实性构成质疑。

## 看多验证信号（出现后应重新审视卖出立场）

1. **长期应收款实质性回落**：2026年半年报或三季报显示长期应收款从30.96亿降至18亿以下，并伴随经营现金流转正——证明垫资模式被有效遏止。
2. **盈利质量改善**：信用减值损失增速放缓或转负，应收账款/收入比率连续两个季度下降，经营现金流/净利润≥1。
3. **算力租赁出现实质性商业化证据**：公司公开披露大客户合同金额、GPU规模和利用率数据，证明其能产生可持续利润（而非仅“2.48亿无贡献”的叙事）。

**这三个信号是直接针对我们看空逻辑核心的高门槛验证点。** 任何一个的单独出现都不足以改变评级，需要至少两个同时满足，且需在估值能消化（即股价已充分回调）的前提下才考虑回补。

## 看空验证 / 下调信号（出现后将强化卖出信心）

1. **长期应收款继续膨胀**：半年报显示长期应收款从30.96亿进一步升至35亿以上——证明垫资换单模式仍在加速。
2. **信用减值损失加速恶化**：半年报信用减值损失超过1亿（全年化3亿以上），且公司解释仍归于“正常业务扩展”——则将直接侵蚀净利润。
3. **经营现金流持续为负**：即使考虑到季节性，2026年Q2经营现金流仍为显著负值——说明这不是季节问题而是结构性问题。
4. **分红被削减或融资计划加速**：公司砍掉分红以保留现金，或短期内密集发行超短融和中票——这将是财务压力的直接确认。

## 数据覆盖度审计

| 关键数据类型 | 状态 | 对评级的冲击 |
| :--- | :--- | :--- |
| 财务报告（年报/季报） | 充分可用 | 支持卖出结论的核心证据（长期应收款、经营现金流、信用减值损失、资产负债率）全部来自此层 |
| 同行业可比公司 | 部分可用（城发环境、中原环保等，但属不同细分） | 限制了相对配置判断的精度，但不影响本股的绝对卖出结论 |
| 海外订单交付和利润转化 | 严重缺失 | 海外5.4亿订单的毛利率、回款周期、本地化利润率均未披露——这是多头“第二增长曲线”叙事中最大的研究鸿沟 |
| 算力租赁商业经济性 | 严重缺失 | GPU型号/数量、利用率、客户、价格——五个最关键变量全部缺失。算力只能作为低概率期权，不能进入任何估值计算 |
| 合同负债季度历史基线 | 不完整 | 单季同比零增长（2025Q1=2026Q1）是强证据，但缺乏多季度季度环比基线限制了我们对“季节模式”的更精细校正 |
| 价格/库存/订单的高频数据 | 无法获得 | 环卫装备的月度销量、中标价格、在手订单额等高频数据不可用——但这对于结论并非关键 |

**证据充分性判断：** 即便存在上述研究鸿沟，现有的财务报告证据（资产负债表断层、经营现金流恶化、信用减值翻倍）已足够支撑卖出评级。长期应收款暴增17.2亿是量化、即时、已发生且来源无误的硬证据，它直接证伪了多头“增长质量良好”的核心假设。缺失的算力租赁数据只会进一步削弱多头的看涨逻辑，而非削弱我们的看跌逻辑。
