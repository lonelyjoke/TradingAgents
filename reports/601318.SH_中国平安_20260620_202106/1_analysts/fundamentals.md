# 投资备忘录：中国平安（601318.SH）

**日期：** 2026年6月20日  
**研究员：** 买方基本面研究

## 一、核心投资论点（Core Thesis）

**核心判断：** 市场因投资端波动和NBV利润率下行而给平安定价了"特许经营权受损"的极端悲观估值（PE处于5年1.5%分位、PB低于1倍），但运营引擎——NBV量增、营运利润、产险承保质量和分红能力——实际在改善。预期差在于：市场低估了量增对利润的弥补能力，以及综合金融+医疗养老生态对客均价值的持续提升。

**投资主线分两层：**
1. **价值修复主线（Base Case）：** 寿险NBV量增（+20.8%）驱动营运利润增长→OPAT推升内含价值→P/EV修复→PB回升至1倍以上
2. **防御分红主线（Income Case）：** 连续14年现金分红增长，FY2025全年分红5.25元/股（含中期），当前股息率约5.5%，有充足的经营现金流覆盖（FY2025 OCF/净利润=4.9x）

## 二、行业周期定位

| 维度 | 判断 | 证据 |

| 利率环境 | 中性偏压 | 长期利率低位压制固收再投资收益率，但权益市场2025年综合投资收益率6.3%为近5年最高 |
| 财产险周期 | 改善 | COR 95.8%，同比优化0.8ppt，新能源车险规模增长 |

**周期定位：** 非传统周期股，但寿险业务处于代理人转型完成后的"量增验证期"。

## 五、预期差分析

**当前市场定价隐含的假设：**
- PE 6.73隐含归母净利润约1,328亿元（FY2025实际为1,348亿，基本匹配）
- PB 0.88隐含市场认为股东权益在缩水或ROE不可持续
- 股价过去1年跌8.2%的同时EPS代理变量上升14.2%，完全是PE压缩（-19.7%）在主导

**我们的预期差在于：**
1. **市场定价了净利下滑趋势**（Q1净利润-7.4%），但营运利润+7.6%才是核心盈利能力的真实体现
2. **市场定价了NBV margin下滑的悲观线性外推**，忽视了量（首年保费+45.5%）对绝对NBV的贡献
3. **市场将保险股归入"红利/防御"标签后遭资金流出**（科技/成长风格极端拥挤时），而非基本面恶化
4. **股息率5.5%在PB<1的背景下提供下行保护**——即使股价持平，年化回报约10.5%来自股息

## 八、估值情景

### Base Case（概率50-60%）
- **假设：** NBV全年增长15-18%，OPAT增长5-8%，归母净利润1,350亿
- **估值：** PE 8x（历史中位数附近）→ 目标价60-63元，上行空间22-28%
- **催化剂：** 半年报NBV持续两位数增长、分红方案确认

### Bull Case（概率20-25%）
- **假设：** NBV加速至25%+、权益市场上涨推升投资收益、NBV margin企稳
- **估值：** PE 10x（仍低于5年均值8.34）→ 目标价75-80元，上行空间52-62%
- **催化剂：** 利率企稳、代理人NBV margin回升、医疗养老变现加速

### Bear Case（概率20-25%）
- **假设：** 权益市场下跌、NBV margin继续恶化至20%以下、投资端出现信用风险
- **估值：** PB 0.8x（5年最低0.77）→ 目标价42-45元，下行空间9-15%
- **触发：** 半年报NBV增速降至个位数、产险COR恶化至98%+

### 证伪条件（Falsification Signals）
1. **NBV增速降至个位数以下** → 量增逻辑破裂
2. **COR恶化至98%以上** → 产险承保利润消失
3. **分红削减或停滞** → 管理层信心丧失
4. **PB跌破0.77（5年最低）且无反弹** → 下行趋势确认
5. **偿付能力充足率降至监管红线附近** → 资本约束

## 十一、管理层与资本配置评估

- **分红纪律：** 连续14年增长，FY2025全年5.25元/股（含中期），FY2024为4.86元，年均复合增长约8%

- **资本配置：** 每年投资现金流-5,600亿（主要对应保险资金配置于债券、权益等），属于业务自然需求而非过度扩张
- **商誉：** 381亿，占总资产0.27%，来源于新方正集团等历史并购，减值风险可控
- **管理层评价：** 马明哲/谢永林长期带队，历史ROE和股东回报记录优秀
